About

wgsl-analyzer is an implementation of Language Server Protocol for the WGSL programming language. It provides features like completion and goto definition for many code editors, including VS Code, Emacs and Vim.

Quick Start

For VS Code, install wgsl-analyzer extension from the marketplace. Prebuilt language server binaries for Windows, Linux and Mac are available on the releases page.

Instructions for other editors, building from source and the list of features are found in the manual.

You can ask questions about using wgsl-analyzer in the wgsl-analyzer discord: wgsl-analyzer .

Support

wgsl-analyzer is a free and open source project, developed with support from individuals. Donate to wgsl-analyzer on Open Collective.
